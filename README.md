# Aadhaar Data Insights Dashboard - Frontend

This is the frontend interface for the Aadhaar Data Insights Dashboard hackathon project.

## Overview
A clean, modern, and responsive web interface designed to present insights on Aadhaar authentication trends, failure rates, and service efficiency.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3 (Vanilla)**: Custom design system with "Government Blue" theme.
- **JavaScript (Vanilla)**: Simple interactivity.
- **Lucide Icons**: For clean visual elements (loaded via CDN).

## Project Structure
- `index.html`: Landing page with project overview, methodology, and team sections.
- `dashboard.html`: Interactive dashboard view with insight cards and chart placeholders.
- `css/styles.css`: Centralized stylesheet with variables for consistent theming.
- `js/main.js`: Logic for navigation and dashboard simulated interactions.

## How to Run
1. Simply double-click `index.html` to open the Landing Page in any modern browser.
2. Click "View Dashboard" to navigate to the analytics view.

## Notes
- The dashboard uses UI placeholders for charts as requested.
- Data shown is dummy/static for presentation purposes.
